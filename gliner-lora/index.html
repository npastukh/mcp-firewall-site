Как пользоваться

Проверка своих запросов

Контур работает на заранее экспортированных результатах. Для нового запроса добавь его в список запросов и перегенерируй JSON одним Python-командой.

1. Выбери пример

Можно кликнуть по готовому кейсу или вставить текст вручную в поле ниже.

2. Нажми “Проверить”

Если такой prompt уже экспортирован, страница покажет span-ы модели и итог firewall.

3. Если кейс новый

Страница подскажет команду, которой нужно обновить JSON для GitHub Pages.
