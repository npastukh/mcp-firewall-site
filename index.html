MCP Firewall
Главная Демонстрационный стенд Аналитический дашборд

Исследовательский проект

Интеллектуальный межсетевой экран для защиты MCP-серверов в агентных системах

Сайт показывает один целостный сценарий: пользовательский запрос интерпретируется в MCP tools/call, проходит через защитный контур и возвращает итоговое решение с понятным объяснением.

Открыть живой стенд Посмотреть аналитику
Основной контур Rules + CatBoost

Базовая проверка каждого запроса по правилам, признакам и итоговой оценке риска.

Углублённая проверка GLiNER + LoRA

Полнотекстовый semantic-stage, который запускается только для подозрительных сценариев.

Результат allow / warn / block

Система возвращает общий итог и объясняет, какой контур повлиял на решение.

Как это работает

Схема одного запроса

Запрос проходит через единый защитный контур и завершается итоговым решением.

01

Пользовательский запрос

Пользователь вводит естественный текст, например запрос на чтение файла или web-fetch вызов.

02

Интерпретация в MCP

Система формирует структурированный tools/call с аргументами для MCP-сервера.

03

Гибридный анализ

Сначала работает быстрый контур Rules + CatBoost, затем при необходимости подключается GLiNER + LoRA.

04

Итоговое решение

Сайт получает один JSON-ответ с интерпретацией, промежуточными результатами и финальным вердиктом.
